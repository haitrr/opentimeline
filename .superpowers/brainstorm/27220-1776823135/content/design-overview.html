Key Design Decisions

  • Drag replaces existing scroll drag — click still jumps to position; drag now creates a time range selection
  • Live map update during drag — orange = points in selected range, grey = outside. Both device tracks shown
  • Popover on release — anchored to the scrubber, to the left, showing only devices that have data in the range
  • Drag handles — orange lines at top/bottom of selection; user can drag either to adjust range after initial select
  • Escape / click-outside — cancels selection, map returns to normal
