# Visit Notes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a nullable markdown-supported notes field to visits, editable in EditVisitModal and rendered on VisitCard when present.

**Architecture:** Extend the Prisma Visit model with `notes String?`, thread it through the API GET/PUT/POST endpoints, add a textarea to EditVisitModal, and render the notes with `react-markdown` on VisitCard below the photos/sub-places section.

**Tech Stack:** Prisma (PostgreSQL), Next.js App Router API routes, React, react-markdown, vitest + @testing-library/react for unit tests, Playwright for E2E.

## Global Constraints

- Font size for inputs/textareas must be ≥ 16px (iOS zoom prevention).
- No live preview in EditVisitModal — rendered output appears on VisitCard after saving.
- Notes are always visible on VisitCard when present (no toggle).
- TDD: write the failing test before the implementation in every task.
- Commit after each task.

- [ ] **Step 3: Update `app/api/visits/route.ts`**

In `GET`: the `prisma.visit.findMany` result already includes `notes` automatically once Prisma schema is updated — no change needed to the select/include. The spread `...v` in the map already passes it through.

In `POST`: extract `notes` from body and pass to `prisma.visit.create`:

```typescript
export async function POST(request: NextRequest) {
  const body = await request.json().catch(() => null);
  if (!body) return NextResponse.json({ error: "Invalid JSON" }, { status: 400 });

  const { placeId, arrivalAt, departureAt, status = "confirmed", notes } = body as {
    placeId?: number;
    arrivalAt?: string;
    departureAt?: string;
    status?: string;
    notes?: string;
  };

  if (!placeId || !arrivalAt || !departureAt) {
    return NextResponse.json({ error: "placeId, arrivalAt, departureAt are required" }, { status: 400 });
  }

  const arrival = new Date(arrivalAt);
  const departure = new Date(departureAt);
  if (isNaN(arrival.getTime()) || isNaN(departure.getTime())) {
    return NextResponse.json({ error: "Invalid date format" }, { status: 400 });
  }
  if (arrival >= departure) {
    return NextResponse.json({ error: "arrivalAt must be before departureAt" }, { status: 400 });
  }

  const visit = await prisma.visit.create({
    data: { placeId, arrivalAt: arrival, departureAt: departure, status, notes: notes ?? null },
    include: { place: { select: { id: true, name: true, lat: true, lon: true, radius: true } } },
  });

  return NextResponse.json(visit, { status: 201 });
}
```

- [ ] **Step 4: Update `app/api/visits/[id]/route.ts` PUT handler**

Extract `notes` from body and include it in the `prisma.visit.update` call. Replace the `data` object in the `updated` call:

```typescript
export async function PUT(
  request: NextRequest,
  { params }: { params: Promise<{ id: string }> }
) {
  const { id } = await params;
  const visitId = parseInt(id, 10);

  if (isNaN(visitId)) {
    return NextResponse.json({ error: "Invalid id" }, { status: 400 });
  }

  const visit = await prisma.visit.findUnique({ where: { id: visitId } });
  if (!visit) {
    return NextResponse.json({ error: "Not found" }, { status: 404 });
  }

  const body = await request.json();

  const nextPlaceId = body.placeId != null ? Number(body.placeId) : visit.placeId;
  if (!Number.isInteger(nextPlaceId)) {
    return NextResponse.json(
      { error: "placeId must be an integer" },
      { status: 400 }
    );
  }

  const place = await prisma.place.findUnique({ where: { id: nextPlaceId } });
  if (!place) {
    return NextResponse.json({ error: "placeId not found" }, { status: 400 });
  }

  const status = body.status != null ? String(body.status) : visit.status;
  if (!["suggested", "confirmed", "rejected"].includes(status)) {
    return NextResponse.json(
      { error: "status must be 'suggested', 'confirmed', or 'rejected'" },
      { status: 400 }
    );
  }

  const arrivalAt = body.arrivalAt != null ? new Date(String(body.arrivalAt)) : visit.arrivalAt;
  const departureAt =
    body.departureAt != null ? new Date(String(body.departureAt)) : visit.departureAt;

  if (Number.isNaN(arrivalAt.getTime()) || Number.isNaN(departureAt.getTime())) {
    return NextResponse.json(
      { error: "arrivalAt and departureAt must be valid ISO datetimes" },
      { status: 400 }
    );
  }

  if (departureAt.getTime() <= arrivalAt.getTime()) {
    return NextResponse.json(
      { error: "departureAt must be after arrivalAt" },
      { status: 400 }
    );
  }

  const notes = "notes" in body ? (body.notes == null ? null : String(body.notes)) : visit.notes;

  const updated = await prisma.visit.update({
    where: { id: visitId },
    data: { placeId: nextPlaceId, status, arrivalAt, departureAt, notes },
  });

  return NextResponse.json(updated);
}
```
